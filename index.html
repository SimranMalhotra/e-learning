Latest Courses

Python

Python is a widely used high-level programming language for general-purpose programming. An interpreted language, Python has a design philosophy which emphasizes code readability. The language provides constructs intended to enable writing clear programs on both a small and large scale.

Details

Computer Graphics

This course give students the basic knowledge of how to develop, design and implement two and three dimensional graphical structures. To enable students to acquire knowledge Multimedia compression and animations and to learn Creation, Management and Transmission of Multimedia objects.

Details

Data Structures through C++

This course makes students familiar with basic techniques of object oriented principles and exception handling using C++. It makkes them familiar with the concepts like inheritence and polymorphism and advanced data structures such as balanced search trees, AVLTrees and B Trees.

Details

Hadoop and Big Data

The ‘Introduction to Big Data and Hadoop’ is an ideal course for individuals who want to understand the basic concepts of Big Data and Hadoop. The course focuses on the basics of Big Data and Hadoop. It further provides an overview of the commercial distributions of Hadoop as well as the components of the Hadoop. 

Details

Finite Element Methods

A finite element method (abbreviated as FEM) is a numerical technique to obtain an approximate solution to a class of problems governed by elliptic partial differential equations. To solve an initial value problem, one needs both the finite element method as well as the finite difference method

Details

Electrical Machines I

This is a basic course on rotating electrical machines. This course covers the topics related to principles, performance, applications and design considerations of dc machines and transformers. This course makes students familiar with the methods of speed control and testing methods of DC motors.
